## 选品分析

| 候选 | 评价 | 结论 |
|------|------|------|
| 待办清单 CLI | 烂大街，无差异化 | ❌ |
| 天气查询 CLI | 依赖 API key，用户需注册 | ❌ |
| **文件整理 CLI** | **刚需 + 有竞品但可差异化** | **✅** |
| 笔记 CLI | 复杂度偏高，一轮做不完 | ❌ |

差异化策略：安全（dry-run）、可回滚（undo）、好看（rich 输出）—— 这些是竞品普遍缺失的。

## 学到的教训

- `setup-cron.sh` 引用的任务文件名是 `tasks.md`（与 protocol 不冲突）
- 第一轮需要同时初始化目录结构和 Git 仓库
