## 经验教训

- **CI badge 风险**: 如果 CI workflow 因 token 问题被移除，badge 立刻变坏链。用测试计数 badge 更稳定。
- **PAT scope 限制**: `workflow` scope 是独立权限，classic PAT 如果没有这个 scope 就无法推送 `.github/workflows/` 目录。
- **WSL 网络**: WSL2 DNS 解析有时不稳定，GitHub API 调用可能间歇性失败。
