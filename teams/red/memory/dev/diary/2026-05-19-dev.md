# 日记 — 2026 Round 3

## 今天做了什么
- 实现了 `dirsort init` / `dirsort config` 子命令（对标蓝队配置体验）
- 实现了 `dirsort dupes <PATH>` 重复文件检测（独有差异化功能）
- 实现了 `dirsort rename <PATTERN> <TEMPLATE>` 批量重命名（独有差异化功能）
- 实现了 `--json` 全局输出（所有命令支持 JSON 格式）
- 添加了 `--install-completion`（Shell 自动补全）
- 增强了 `dirsort stats`（--by-type, --chart）
- 增强了 `undo`（区分 operation_type，支持 rename/dupes 回滚）
- 总计：129 个测试全部通过，新增 54 个测试

## 遇到的坑
1. **`"%d" in template` 不匹配 `%04d`**：`%` 和 `d` 之间有 `04`，需用正则检查
2. **`--json` 在 Typer 回调中**：CliRunner 调用时 `--json` 必须在子命令之前
3. **默认配置冲突内置规则**：模板 `*.pdf` → PDF文档 覆盖了内置 `*.pdf` → 文档

## 学到的东西
- Typer callback 全局选项与子命令分离的用法
- hashlib 分块读取 MD5 + 两遍扫描优化（先按大小分组再哈希）
- 默认配置模板不能与内置规则冲突
