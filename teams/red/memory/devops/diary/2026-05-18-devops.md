## 学到的东西

- WSL 下通过 cmd.exe/c 调用 PowerShell 可以执行需要 Windows 网络栈的操作
- Git push 时 token 嵌入 URL 的格式：`https://user:token@github.com/owner/repo.git`
- GitHub Create Release API 需要 `tag_name`、`name`、`body` 三个字段
